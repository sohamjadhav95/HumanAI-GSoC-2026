This repo contains my work for the HumanAI GSoC 2026 evaluation. HumanAI builds AI tools for historical document analysis think digitising manuscripts, OCR on degraded archival sources, that kind of thing.

The evaluation project (OCR1) asked me to build an OCR pipeline for 17th-century Spanish printed texts, which break every modern OCR tool in predictable ways. I built the full system from scratch PDF ingestion through to LLM-corrected transcript and documented exactly what worked and what didn't.
